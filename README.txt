Cloud Poodll Anywhere
==========

The Cloud Poodll plugin for TinyMCE.
It allows the user to record(or upload) audio and video directly into forum posts, assignment descriptions, page resource content, question descriptions, question responses and other areas.

Install PoodLL anywhere by unzipping it and putting the "cloudpoodll" folder in
[moodle]/lib/editors/tinymce/plugins  then visit your site
administration -> notifications page and follow the prompts for Moodle to install it.
